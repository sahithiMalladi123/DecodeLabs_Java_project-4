# Currency Converter
A Java-based Currency Converter that allows users to convert amounts between INR, USD, and EUR using predefined exchange rates through a simple console-based int

## Features
- Convert between INR, USD, and EUR
- User-friendly menu-driven interface
- Fast currency conversion
- Input validation for currency selection
- Console-based application

## CONCEPTS USED:
- Scanner Class
- Switch Statements
- Arithmetic Operations

## Supported Currencies
1. INR (Indian Rupee)
2. USD (US Dollar)
3. EUR (Euro)

## Project Structure
- `DecodeLabs_java_p4.java` (Main Program File)

## How to Run


### Steps

1. Clone the repository:
git clone <repository-url>

2. Navigate to the project folder:

3. Compile the Java file:
```bash
javac DecodeLabs_java_p4.java
```

4. Run the program:
```bash
java CurrencyConverter
```
